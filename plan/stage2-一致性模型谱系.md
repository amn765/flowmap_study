# 阶段 2 · Consistency Models 谱系（约 3-4 天）

> 目标：吃透流映射方法族里最有名的前身 —— 一致性模型（Consistency Models, CM）及其轨迹版（CTM）。理解「把半群/自洽变成损失」的第一个成功范式。这能让你在阶段 4 看 MeanFlow 时立刻分辨「它继承了什么、改了什么」。
> （梯度提示：在阶段 1 的流映射公理上，加一层「怎么变成可训练损失」。仍以理解为主，不强求复现。）

主线资料：[Consistency Models（Song et al., 2023）](https://arxiv.org/abs/2303.01469) + [Consistency Trajectory Models / CTM（Kim et al., 2024）](https://arxiv.org/abs/2310.02279) + 一篇中文精读博客辅助。

## 任务清单

### ☐ 2-1 一致性函数：单时间流映射
- CM 学一个**一致性函数** `f_θ(x_t, t)`，把轨迹上**任意**时刻的点直接映回轨迹起点（数据端）：`f_θ(x_t, t) ≈ x_0`。
- 这是流映射的一个特例：终点时间固定为 0，只剩单时间参数 t。一致性 = 同一条轨迹上所有点映到同一个 `x_0`。
- 思考题：CM 把「双时间映射」砍成「映到固定终点 0」。这样做省了什么、又丢了什么灵活性？（提示：能不能从中间停一站？）
- **完成标志**：能说清一致性函数的输入输出，以及它是流映射的哪种退化。

### ☐ 2-2 边界条件与一致性损失
- 边界条件（必须满足，否则解退化）：在起点 `t = ε` 处 `f_θ(x_ε, ε) = x_ε`。常用参数化 `f_θ = c_skip(t) x + c_out(t) F_θ(x,t)` 自动满足。
- 一致性损失：取轨迹上相邻两点 `x_{t_{n+1}}` 和 `x_{t_n}`，要求它们经 f 映到同一处：
  ```
  L = d( f_θ(x_{t_{n+1}}, t_{n+1}),  f_θ⁻(x_{t_n}, t_n) )
  ```
  其中 `f_θ⁻` 是 stop-gradient 的「目标网络」（EMA 或 detach），`d` 是距离（L2 / LPIPS）。
- 思考题：为什么目标那一支要 stop-gradient？如果两支都回传梯度会怎样？（提示：会塌缩成平凡解。）
- **完成标志**：能写出一致性损失的结构，并解释边界条件和 stop-gradient 各自防止什么。

### ☐ 2-3 ⭐ CD（蒸馏）vs CT（从零训练）
- 同一个一致性损失，相邻点 `x_{t_n}` 怎么来，决定了两种范式：

| 范式 | 相邻点来源 | 是否需要预训练教师 | 关键词 |
|------|------------|--------------------|--------|
| Consistency Distillation (CD) | 用预训练扩散/ODE 求解器走一步得到 | 需要教师 | 蒸馏 |
| Consistency Training (CT) | 用对真实数据加噪的解析公式估计 | 不需要教师 | 从零 |
- CT 的意义重大：**不依赖教师模型**就能从零训练出少步生成器。这正是 MeanFlow 追求的「from scratch」路线。
- 思考题：CT 为什么能避开教师？它用什么替代了「教师走一步」提供的轨迹信息？
- **完成标志**：能讲清 CD 与 CT 的唯一关键差异，并说明为何 CT 的「from scratch」更受推崇。

### ☐ 2-4 Consistency Trajectory Models（CTM）：双时间
- CTM 把单时间补回双时间：学 `G_θ(x_t, t, s)`，预测从 t 跳到任意中间时刻 s 的结果（不再强制终点为 0）。
- 好处：支持「任意步数」的灵活采样（一步、几步、多步可调），并能与对抗/去噪损失结合提质量。
- 思考题：CTM 的 `G(x_t, t, s)` 和阶段 1 的 `Φ_{t→s}(x_t)` 是不是一回事？差别在哪？
- **完成标志**：能说清 CTM 相比 CM 多了哪个自由度，以及它为什么更接近「完整流映射」。

### ☐ 2-5 改进版一致性（sCM / 连续时间）简述
- 了解趋势即可：早期 CM/CT 用**离散**时间步，对步数划分敏感、训练不稳。后续工作（如连续时间一致性、[sCM](https://arxiv.org/abs/2410.11081)）转向**连续时间**极限，把相邻两点的差分换成微分（导数），训练更稳、更易 scale。
- 这一步「离散差分 → 连续微分」的转变，和 MeanFlow 用**全导数 + JVP**而非有限差分，是同一种品味。先记住这个动机，阶段 5 会真正用到。
- 思考题：差分 `(f(t+Δ) - f(t))/Δ` 取极限就是导数。在实现上，连续时间版要算什么量来代替「走一小步」？（提示：方向导数 / JVP。）
- **完成标志**：能说清「离散一致性 → 连续一致性」改了什么，以及它和 MeanFlow 实现风格的呼应。

### ☐ 2-6 自测：口述题
合上资料，能流畅回答：
1. 一致性损失为什么必须有边界条件和 stop-gradient？
2. CD 与 CT 的唯一关键差异是什么？为什么 CT 更被看重？
3. CTM 相比 CM 多了什么自由度？连续时间版相比离散版改了什么？

## 完成标志
- 能把「一致性 = 自洽约束当损失」这条主线讲清楚，并区分 CD/CT/CTM。
- 理解「离散差分 → 连续微分」的动机，为 MeanFlow 的全导数实现埋好伏笔。
