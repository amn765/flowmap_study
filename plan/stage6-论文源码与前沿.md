# 阶段 6 · 论文精读 · 源码 · 复现与前沿（约 5-7 天）

> 目标：从「玩具能跑」走到「读懂真实工程 + 看清研究脉络」。逐行精读 MeanFlow 论文、通读官方源码、在 MNIST/CIFAR 量级复现一步生成，并把它放回流映射谱系做横向对比，最后扫一遍后续工作与开放问题。学完你具备「做改进 / 写相关工作」的能力。
> （梯度提示：前面阶段已把原理和最小实现打通，这里只是把规模和严谨度往上抬一格。）

主线资料：MeanFlow 论文全文 + 官方代码仓库 + 阶段 2/3 的对比论文。

## 任务清单

### ☐ 6-1 精读 MeanFlow 论文全文
- 逐节读，重点：方法节的每个公式都对照阶段 4/5 验证；实验节看它在 ImageNet 上的 FID / NFE 曲线、消融（r=t 比例、加权、CFG）。
- 边读边记：哪些是必要设计、哪些是工程 trick、哪些是为 scale 服务。
- 思考题：论文宣称「一步 FID 接近多步扩散」，它最关键的 1~2 个设计是什么？去掉会怎样（看消融）？
- **完成标志**：能复述论文主结果与 3 个以上消融结论，指出哪些设计不可或缺。

### ☐ 6-2 ⭐ 通读官方源码
- 把代码和你阶段 5 的骨架对齐，重点定位四处：
  1. `jvp` / 全导数那一行（对应恒等式的 `du/dt`）。
  2. `(r, t)` 采样与 `r=t` 比例处。
  3. 自适应损失加权实现。
  4. CFG 如何融入训练与采样。
- 思考题：源码里有没有你在玩具实现里忽略的稳定性细节（如时间分布、归一化、EMA、预处理）？记下 3 个。
- **完成标志**：能在真实代码里指出上述四处的具体位置，并解释与论文公式的对应。

### ☐ 6-3 在 MNIST/CIFAR 复现一步生成
- 把阶段 5 的 2D 脚本换成卷积/UNet 骨干，在 MNIST（先）或 CIFAR-10（后）上训练，做一步采样，观察样本质量与（可选）FID。
- 控制好算力：先小模型小步数跑通流程，再视资源加大。
- 思考题：从 toy 到图像，哪些超参最敏感（学习率、`r=t` 比例、加权幂次、时间分布）？
- **完成标志**：能一步采样出可辨认的 MNIST 数字（或 CIFAR 上有结构的样本），并记录关键超参。

### ☐ 6-4 横向对比实验
- 在同一数据/骨干下，对比三条线：多步 flow matching、一致性/shortcut（任选一个你阶段 2-3 学过的）、MeanFlow。
- 比较维度：NFE-质量曲线、训练稳定性、实现复杂度。做成一张你自己的对比表。
- 思考题：MeanFlow 的优势在「极少 NFE」区间最明显吗？在多步区间它还领先吗？为什么？
- **完成标志**：能给出一张自测得到的「方法 × NFE × 质量」对比表，并解释趋势。

### ☐ 6-5 前沿延伸与开放问题
- 调研 MeanFlow 之后的发展：与一致性模型的统一理解、更大规模/文生图上的应用、与其他一步方法（蒸馏类、GAN 类）的关系、训练成本与稳定性改进。
- 列出 3 个你认为值得做的开放问题（例如：非线性路径下的恒等式、平均速度的高阶推广、与 SDE/随机采样的结合）。
- 思考题：如果让你在 MeanFlow 上做一个小改进课题，你的切入点和最小验证实验是什么？
- **完成标志**：能讲清 MeanFlow 在当前一步生成版图中的位置，并提出 3 个有依据的研究方向。

### ☐ 6-6 自测与总结
合上资料，能从头到尾讲一遍：
1. 从 flow matching 的 NFE 问题，到流映射，到平均速度，到 MeanFlow 恒等式与一步采样 —— 一条逻辑链。
2. MeanFlow 与一致性 / Shortcut / FMM 的异同各一句话。
3. 写一份半页纸的「MeanFlow 学习总结 + 我的研究想法」。

## 完成标志
- 能像作者一样讲清 MeanFlow 的动机、推导、实现与定位，并有自己的复现数据支撑。
- 产出一份个人总结笔记 + 至少一个可执行的改进 idea，标志整套学习闭环完成。
