# 阶段 1 · 流映射的数学基础（约 3-4 天）

> 目标：把流映射从「直觉」落到「公式」。能写出双时间流映射的定义、半群性质的微分形式，并清楚它与速度场互相导出的关系。这是后面 MeanFlow 恒等式推导的地基。
> （梯度提示：只比阶段 0 多了「把话写成方程」，用到的微积分不超过链式法则和微积分基本定理。）

主线资料：[Flow Map Matching（Boffi et al., 2024）](https://arxiv.org/abs/2406.07507) 第 2-3 节 + 任一 neural ODE / probability flow ODE 的复习材料。

## 任务清单

### ☐ 1-1 ODE 流与求解算子 Φ
- 给定速度场 `v(x, t)`，ODE `dx/dτ = v(x, τ)`，初值 `x_s` 在时刻 s。定义求解算子：
  ```
  Φ_{s→t}(x_s) = x_s + ∫_s^t v(x_τ, τ) dτ = x_t
  ```
- `Φ` 把「初始点 + 起止时间」映射到终点。固定 s、把 t 当变量，`Φ_{s→t}(x_s)` 就是那条轨迹。
- 思考题：为什么说求解算子比速度场「信息更浓缩」？（学到 Φ 就不用再积分了。）
- **完成标志**：能写出 Φ 的积分定义，并解释每个下标的含义。

### ☐ 1-2 ⭐ 双时间流映射与半群性质
- 流映射有**两个时间参数**：起点时间 s 和终点时间 t。记 `Φ_{s→t}(x)`，满足：
  - 自映射（边界）：`Φ_{t→t}(x) = x`。
  - 半群（可组合）：`Φ_{t→u}(Φ_{s→t}(x)) = Φ_{s→u}(x)`。
- 这两条是流映射的「公理」。任何声称学到流映射的网络都必须满足它们。
- 思考题：把半群性质对 t 求导、再令两段时间间隔趋于 0，会得到什么微分关系？（这是 1-4 的引子。）
- **完成标志**：能默写自映射与半群两条性质，并解释它们对应「不动」和「可拼接」两种直觉。

### ☐ 1-3 Lagrangian 与 Eulerian 两种视角
- 同一个流映射可以从两个角度刻画，后面 FMM 和 MeanFlow 的不同损失正源于此：

| 视角 | 跟踪对象 | 直觉 | 对应损失风格 |
|------|----------|------|--------------|
| Lagrangian | 跟着一个粒子走 | 「这颗粒子从 s 到 t 去了哪」 | 直接回归终点位置 |
| Eulerian | 盯住空间中一点看场 | 「在 (x,t) 处场长什么样」 | 用微分/PDE 自洽 |
- 思考题：Lagrangian 需要知道整条轨迹的终点，Eulerian 只需局部微分。哪种更容易做成「不依赖教师模型」的从零训练？
- **完成标志**：能用一句话区分两种视角，并各举一个它们对应的训练思路。

### ☐ 1-4 流映射满足的微分关系
- 把半群性质做无穷小展开，得到流映射的**微分刻画**。固定起点时间 s，对终点时间 t 求导：
  ```
  ∂_t Φ_{s→t}(x) = v( Φ_{s→t}(x), t )
  ```
  即「终点随 t 移动的速度 = 终点处的瞬时速度场」。
- 对起点时间 s 求导则得到另一条（沿特征线的）关系。这两条微分方程 + 边界 `Φ_{t→t}=x` 完全确定了流映射。
- 思考题：上式右边的 `v` 在「终点」处取值。如果网络直接输出 Φ，怎么把这条 PDE 变成可优化的残差？（Eulerian 损失的核心。）
- **完成标志**：能写出 `∂_t Φ = v(Φ, t)` 并解释它就是「ODE 的算子形式」。

### ☐ 1-5 流映射与瞬时速度场互相导出
- 两个方向都要会：
  - 速度 → 映射：积分 `Φ_{s→t} = x + ∫_s^t v dτ`。
  - 映射 → 速度：求导，`v(x, t) = ∂_t Φ_{s→t}(x) |_{s=t}`（在 s=t 处对 t 求导即还原速度）。
- 这条「映射↔速度」的桥，正是 MeanFlow「平均速度」恒等式的雏形 —— 平均速度就是把 `∫ v` 除以时间区间。
- 思考题：如果定义 `u(x,s,t) = Φ_{s→t}(x) - x` 除以 `(t-s)`，它是什么的平均？（剧透：这就是阶段 4 的主角。）
- **完成标志**：能在「积分得映射」「求导得速度」两个方向自由切换。

### ☐ 1-6 自测：口述题
合上资料，能流畅回答：
1. 写出双时间流映射的自映射与半群性质。
2. `∂_t Φ_{s→t}(x) = ?`，右边为什么在终点处取速度？
3. Lagrangian 与 Eulerian 视角分别对应什么训练思路？

## 完成标志
- 能不看资料推出 `∂_t Φ = v(Φ, t)`，并解释它与半群性质的关系。
- 理解「积分 ↔ 求导」这座桥，为阶段 4 的平均速度做好准备。
