# 阶段 0 · 从速度场到流映射：动机与全景（约 2-3 天）

> 目标：搞清楚「为什么有了 flow matching 还要 flow map」。你已经会用速度场 + ODE 多步采样，这一阶段要把视角从「学瞬时速度」切换到「直接学时间到时间的映射」，并在脑子里建立整个流映射方法族的地图。
> （梯度提示：这一阶段不推公式，只对齐动机和术语，为后面所有阶段铺路。）

主线资料：[MeanFlow 论文（Mean Flows for One-step Generative Modeling, Geng et al., 2025）](https://arxiv.org/abs/2505.13447) 的引言与相关工作章节 + [Flow Map Matching（Boffi, Albergo, Vanden-Eijnden, 2024）](https://arxiv.org/abs/2406.07507) 的引言。

## 任务清单

### ☐ 0-1 重温 flow matching 的采样瓶颈
- 回忆你已经掌握的：flow matching 学一个速度场 `v_θ(x, t)`，采样时解 ODE `dx/dt = v_θ(x, t)`，从噪声积分到数据。
- 关键痛点：数值积分需要多步，每步一次网络前向 —— 即 NFE（number of function evaluations）大。一张高质量图常要几十到上百 NFE。
- 思考题：CFM / rectified flow 把路径拉直，是在「减少 NFE」这个目标上做文章。但即使路径再直，瞬时速度场本质上仍要积分。能不能干脆**跳过积分**？
- **完成标志**：能一句话说清「flow matching 慢在哪、为什么减步数难」。

### ☐ 0-2 ⭐ 流映射（flow map）是什么：求解算子视角
- 核心切换：不学瞬时速度 `v(x, t)`，而学**流映射** `Φ_{s→t}(x)` —— 它直接把 `s` 时刻的点 `x_s` 送到 `t` 时刻的点 `x_t`，等价于「一次性解完从 s 到 t 的 ODE」。
- 它就是 ODE 的**求解算子**（solution operator）：`x_t = Φ_{s→t}(x_s)`。如果能学到 `Φ_{1→0}`（噪声 t=1 → 数据 t=0），一次前向就出图，NFE = 1。
- 思考题：速度场是流映射的什么？（提示：流映射对终点时间求导，在 s=t 处就退化回速度。）
- **完成标志**：能用「求解算子」一句话定义流映射，并说清它与速度场的直觉关系。

### ☐ 0-3 半群性质与一致性
- 流映射天然满足**半群性质**：`Φ_{t→u} ∘ Φ_{s→t} = Φ_{s→u}` —— 先走 s→t 再走 t→u，等于直接 s→u。
- 这条性质是所有流映射方法的训练抓手：网络如果真学到了流映射，就必须自洽（self-consistent）。「一致性模型」「自蒸馏」「shortcut」本质都在逼网络满足这条约束。
- 思考题：为什么半群性质能当「免费的监督信号」？它和真实数据有没有关系？
- **完成标志**：能默写半群性质，并说清它为什么能转成损失函数。

### ☐ 0-4 流映射方法族全景地图
- 在动手推导前，先建立地图。下面这些方法都在学某种形式的流映射：

| 方法 | 学什么 | 时间条件 | 训练方式 |
|------|--------|----------|----------|
| Consistency Models (CM) | `f(x_t, t) → x_0` | 单时间 | 蒸馏(CD) / 从零(CT) |
| Consistency Trajectory (CTM) | `G(x_t, t, s)` | 双时间 | 蒸馏为主 |
| Shortcut Models | `v(x_t, t, d)` 含步长 d | 单时间+步长 | 从零自一致 |
| Flow Map Matching (FMM) | `Φ_{s→t}` | 双时间 | 蒸馏-free 统一框架 |
| MeanFlow | 平均速度 `u(x, r, t)` | 双时间 | 从零，恒等式监督 |
- 思考题：这些方法的差异主要在三个轴上 —— 单时间 vs 双时间、蒸馏 vs 从零、学映射本身 vs 学平均速度。MeanFlow 在每个轴上选了哪一边？
- **完成标志**：能凭记忆画出这张表的「方法 + 学什么」两列。

### ☐ 0-5 自测：口述题
合上资料，能流畅回答：
1. 一句话：flow map 相比 flow matching 解决了什么问题？
2. 半群性质是什么？为什么它能当训练信号？
3. 在全景表里，MeanFlow 和 Consistency Models 最大的区别是什么？
