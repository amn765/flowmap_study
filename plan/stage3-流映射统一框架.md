# 阶段 3 · 流映射统一框架与 Shortcut（约 3-4 天）

> 目标：站到「流映射统一框架」的高度，看清 Consistency / Shortcut / MeanFlow 都是同一棵树上的分支。重点掌握 Flow Map Matching（FMM）的 Lagrangian / Eulerian 两类目标，以及 Shortcut Models 的「步长条件 + 自一致」。学完你就能给任意一个新流映射方法「归类定位」。
> （梯度提示：把阶段 2 的具体方法抽象成统一损失家族，是阶段 4 直接接 MeanFlow 的最后一块拼图。）

主线资料：Flow Map Matching（Boffi, Albergo, Vanden-Eijnden, 2024）+ One Step Diffusion via Shortcut Models（Frans et al., 2024）。

## 任务清单

### ☐ 3-1 Flow Map Matching：统一框架
- FMM 直接以**双时间流映射** `X_{s,t}(x)`（= 阶段 1 的 `Φ_{s→t}`）为学习对象，给出一套不依赖教师的训练目标，把一致性模型等纳为特例。
- 核心思想：流映射受两条约束 —— 边界 `X_{t,t}(x)=x` 和微分关系 `∂_t X_{s,t} = v(X_{s,t}, t)`。把这两条变成可优化的回归 / 残差，就得到损失。
- 思考题：FMM 说「CM 是我的特例」，具体是把 `X_{s,t}` 的哪个时间固定后得到 CM？
- **完成标志**：能说清 FMM 学的对象，以及它靠哪两条约束构造损失。

### ☐ 3-2 ⭐ Lagrangian 与 Eulerian 目标
- 同一个流映射，两类损失（呼应阶段 1-3）：

| 目标 | 形式（直觉） | 需要什么 | 特点 |
|------|--------------|----------|------|
| Lagrangian | 让 `X_{s,t}` 回归「沿真实速度积出来的终点」 | 短程积分/教师轨迹 | 直接、但可能要积分 |
| Eulerian | 让 `∂_t X_{s,t} - v(X_{s,t}, t)` 残差为 0 | 只需局部微分 + 自洽 | 蒸馏-free、可从零 |
- Eulerian 目标里的 `v(X, t)` 可以用网络自己在 s=t 处的输出提供（自洽），于是完全不需要教师 —— 这就是「自蒸馏 / distillation-free」。
- 思考题：Eulerian 残差里要算 `∂_t X_{s,t}`，这是对**网络输出**关于时间求导。实现上用什么工具高效拿到它？（提示：又是 JVP，阶段 5 会用。）
- **完成标志**：能写出两类目标的差别，并解释 Eulerian 为何能做到「不依赖教师」。

### ☐ 3-3 Eulerian 自蒸馏 / 渐进式自举
- 「自蒸馏」常见实现：用网络在**相邻/更短区间**上的输出（stop-gradient）当作监督，逼网络在**更长区间**上自洽 —— 等价于半群性质的可微版本。
- 渐进式（progressive）：先学短跳，再逐步学长跳，难度逐级上升，避免一上来就学「一步到位」的最难映射。
- 思考题：这种「短跳监督长跳」和阶段 2 一致性损失「相邻点映同一处」在数学上是不是同源？
- **完成标志**：能描述自蒸馏怎么用网络自身当老师，以及渐进式为什么训练更稳。

### ☐ 3-4 Shortcut Models：步长条件 + 自一致
- Shortcut 用**单个**网络 `s_θ(x_t, t, d)`，多输入一个**步长** d，预测在 (x,t) 处走步长 d 的「捷径方向」。
- 自一致约束（核心）：走一步大步 `d` 应等于连走两小步 `d/2`：
  ```
  s_θ(x_t, t, 2d) ≈ 0.5 * [ s_θ(x_t, t, d) + s_θ(x_{t+d}, t+d, d) ]    (stop-grad 右侧)
  ```
  当 `d → 0`，退化为普通 flow matching 速度。
- 它和 MeanFlow 极像：都用「区间长度」当条件、都靠自洽从零训练。差别在 MeanFlow 用**平均速度 + 解析恒等式**，无需「连走两步」的离散自举。
- 思考题：Shortcut 的步长 d 和 MeanFlow 的区间 `(t-r)` 是同一个角色吗？MeanFlow 用什么替代了 Shortcut 的「2 小步拼 1 大步」？
- **完成标志**：能写出 Shortcut 的自一致约束，并指出它与 MeanFlow 的相似与关键区别。

### ☐ 3-5 ⭐ 方法对照总表
- 把谱系收拢成一张你能默写的表（按三个轴对比）：

| 方法 | 学习对象 | 时间条件 | 自洽信号 | 是否需教师 |
|------|----------|----------|----------|------------|
| CM / CT | `f(x_t,t)→x_0` | 单时间 | 相邻点映同一终点 | CT 否 / CD 是 |
| CTM | `G(x_t,t,s)` | 双时间 | 轨迹一致 + 对抗 | 多为是 |
| Shortcut | `s(x_t,t,d)` | 单时间+步长 | 1 大步 = 2 小步 | 否 |
| FMM (Eulerian) | `X_{s,t}` | 双时间 | 微分残差自洽 | 否 |
| MeanFlow | 平均速度 `u(x,r,t)` | 双时间 | 平均速度恒等式 | 否 |
- **完成标志**：能凭记忆复现这张表的「学习对象 / 自洽信号 / 是否需教师」三列。
